Curso Práctico de Desarrollo Frontend

Ruta: HTML y CSS a profundidad

Profesora: Estefany Aguilar

Objetivo del curso: Aprender desarrollo frontend con responsive design, poniendo en práctica maquetación con HTML y CSS para construir diferentes ejemplos de páginas y componentes de un e-commerce profesional. Como utilizar variables de CSS para los colores, fuentes y tamaños de tu aplicación. Crear diferencias en la vista de la página dependiendo del tamaño del dispositivo, y finalmente, aprender a identificar las vistas y componentes como si fuera un desarrollo real en una empresa de tecnología.

Algunos contenidos vistos:

  • El uso de :root { } para crear variables que se utilizan en CSS.
  • Utilizar grid y grid-template-rows para crear tablas con asignaciones automaticas y fraccionadas.
  • Utilizar flex para colocar un elemento al lado del otro y flex-direction en forma de columnas para que se ordenen de arriba hacia abajo.
  • Crear media queries para dispositivos móviles (Cambiar la resolución a menos de 640 pixeles para verlo).
  • Utilizar gap para agregar espacio entre filas y columnas
  • Utilizar nth-child para heredar estilos CSS de padre a hijos.
  • Utilizar !important y cuando conviene hacerlo.
